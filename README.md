# Facial Emotion Detection using Deep Learning

## Overview
This project detects human facial emotions using a CNN model built with TensorFlow/Keras and OpenCV.

## Features
- Emotion Detection
- CNN Model Training
- Live Webcam Detection
- Single Image Prediction

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy

## Dataset
FER2013 Dataset
